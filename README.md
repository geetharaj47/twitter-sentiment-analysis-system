# Technologies

This project integrates modern **machine learning and backend technologies**.

### Machine Learning
- PyTorch
- HuggingFace Transformers
- BERT (Bidirectional Encoder Representations from Transformers)

### Data Processing
- Pandas
- NumPy
- Scikit-learn

### Backend API
- FastAPI
- Uvicorn

### Frontend Interface
- Streamlit

# Features

### Real-Time Sentiment Prediction
Predicts tweet sentiment instantly.

### Transformer-Based NLP
Uses **BERT transformer architecture** for accurate classification.

### REST API
FastAPI backend for model inference.

### Interactive UI
Streamlit dashboard for testing predictions.

### Multiple Models
Includes:
- Baseline Model
- LSTM Model
- BERT Transformer Model
